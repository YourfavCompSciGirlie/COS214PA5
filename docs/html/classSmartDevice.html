Detailed Description

Context class in the State Pattern that represents a smart device.

The SmartDevice class is the context that interacts with various device states (e.g., OnState, OffState). It allows for state transitions and delegates actions to the active state. This class provides common functionality for all smart devices by inheriting from the Device class.
