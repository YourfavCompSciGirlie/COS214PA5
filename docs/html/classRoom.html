Detailed Description

Composite class representing a room containing smart devices.

The Room class acts as a composite in the Composite design pattern. It allows adding, removing, and managing smart devices within the room. The Room class inherits from the Device class, meaning that actions can be performed on it as well as on the devices it contains.
