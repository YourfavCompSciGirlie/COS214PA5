Detailed Description

The HumiditySensor class, a concrete subject that detects and notifies devices about humidity changes.

This class inherits from the Sensor class and represents a humidity sensor. It maintains a list of registered devices (observers) and notifies them when humidity levels change, triggering specific actions.
