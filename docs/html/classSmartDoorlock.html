Detailed Description

A concrete class representing a smart door lock device.

The SmartDoorlock class inherits from SmartDevice and represents a single smart door lock. It implements methods for performing actions and retrieving the current status based on the state pattern.

SmartDoorlock::SmartDoorlock ( )

Construct a new SmartDoorlock object.

The constructor initializes the smart door lock with an unlocked state.

Initializes a new SmartDoorlock device and sets its initial state to "unlocked" by associating it with the UnlockedState object. The device can transition between locked and unlocked states using the performAction method.
