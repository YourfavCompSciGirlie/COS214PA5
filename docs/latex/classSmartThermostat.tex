\doxysubsection{Detailed Description}
A concrete class representing a smart thermostat device. 

The \mbox{\hyperlink{classSmartThermostat}{Smart\+Thermostat}} class inherits from \mbox{\hyperlink{classSmartDevice}{Smart\+Device}} and represents a single smart thermostat. It implements methods for performing actions and retrieving the current status based on the state pattern.
