\doxysubsection{Detailed Description}
A concrete class representing a smart door lock device. 

The \mbox{\hyperlink{classSmartDoorlock}{Smart\+Doorlock}} class inherits from \mbox{\hyperlink{classSmartDevice}{Smart\+Device}} and represents a single smart door lock. It implements methods for performing actions and retrieving the current status based on the state pattern.
