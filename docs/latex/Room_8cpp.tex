Implementation of the \mbox{\hyperlink{classRoom}{Room}} class. 

This file contains the implementation of the methods defined in \mbox{\hyperlink{Room_8h}{Room.\+h}}, providing functionality for managing smart devices within a room as part of a composite structure. The \mbox{\hyperlink{classRoom}{Room}} class serves as the composite in the Composite design pattern, allowing actions and state management for multiple devices within the room.
