\doxysubsection{Detailed Description}
Concrete state representing the \char`\"{}\+Cooling\char`\"{} state of a smart device. 

The \mbox{\hyperlink{classCoolingState}{Cooling\+State}} class is responsible for handling actions when a device is currently cooling. It implements the perform\+Action and get\+Status methods for this specific state.
