\doxysubsection{Detailed Description}
Abstract base class for all devices in the composite structure. 

The \mbox{\hyperlink{classDevice}{Device}} class serves as the Component in the Composite Pattern, providing common functionality for both leaf and composite nodes. It defines abstract methods for performing actions, retrieving status, and managing device types. Additionally, it provides virtual methods for adding, removing, and retrieving child devices, which can be overridden by composite classes (e.\+g., \mbox{\hyperlink{classRoom}{Room}}, \mbox{\hyperlink{classHouse}{House}}).
