\doxysubsection{Detailed Description}
Class representing a macro routine that can execute a series of Devices. 

This class allows users to create macros composed of multiple \mbox{\hyperlink{classDevice}{Device}} procedures, providing functionality to add, remove, execute, save, and load macros.
