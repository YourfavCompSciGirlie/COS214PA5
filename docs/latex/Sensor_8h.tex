\doxysubsection{Detailed Description}
Abstract Subject class in the Observer pattern. 

The \mbox{\hyperlink{classSensor}{Sensor}} class acts as a base class for concrete sensor types (e.\+g., motion sensor, humidity sensor). It maintains a list of devices (observers) that are interested in receiving updates when the sensor state changes.
