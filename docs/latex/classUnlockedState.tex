\doxysubsection{Detailed Description}
Concrete state representing the \char`\"{}\+Unlocked\char`\"{} state of a smart device. 

The \mbox{\hyperlink{classUnlockedState}{Unlocked\+State}} class is responsible for handling actions when a device is currently unlocked. It implements the perform\+Action and get\+Status methods for this specific state.
