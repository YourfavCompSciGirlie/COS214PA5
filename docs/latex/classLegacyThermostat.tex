\doxysubsection{Detailed Description}
Represents a legacy thermostat device. 

The \mbox{\hyperlink{classLegacyThermostat}{Legacy\+Thermostat}} class manages the temperature setting for a legacy thermostat. It contains methods for setting and retrieving the temperature.
