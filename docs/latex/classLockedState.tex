\doxysubsection{Detailed Description}
Concrete state representing the \char`\"{}\+Locked\char`\"{} state of a smart device. 

The \mbox{\hyperlink{classLockedState}{Locked\+State}} class is responsible for handling actions when a device is currently locked. It implements the perform\+Action and get\+Status methods for this specific state.
