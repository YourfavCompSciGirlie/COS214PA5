Concrete Subject class that represents a motion sensor in an Observer pattern. 

The \mbox{\hyperlink{classMotionSensor}{Motion\+Sensor}} class is responsible for detecting motion and notifying all registered devices (observers) of any changes in motion state (e.\+g., detected or clear).
