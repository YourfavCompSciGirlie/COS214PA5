\doxysubsection{Detailed Description}
Represents a composite of rooms in a smart house. 

The \mbox{\hyperlink{classHouse}{House}} class serves as a composite element in the design pattern, allowing for the management and operation of multiple rooms. Each room can contain various devices, and the \mbox{\hyperlink{classHouse}{House}} class provides methods to add, remove, and perform actions on these rooms collectively.
