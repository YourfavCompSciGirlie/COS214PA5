\doxysubsection{Detailed Description}
Header file for the \mbox{\hyperlink{classRoom}{Room}} class, part of the Composite design pattern. 

The \mbox{\hyperlink{classRoom}{Room}} class represents a composite entity that contains a collection of smart devices. It allows actions to be performed on multiple devices within the room and manages their states.
