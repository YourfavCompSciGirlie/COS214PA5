\doxysubsection{Detailed Description}
Implementation of the \mbox{\hyperlink{classHouse}{House}} class. 

This file contains the implementation of the methods defined in \mbox{\hyperlink{House_8h}{House.\+h}}, providing functionality for managing rooms as part of a composite structure in a smart house. The \mbox{\hyperlink{classHouse}{House}} class serves as the composite in the Composite design pattern, allowing actions and state to be applied across multiple rooms.
